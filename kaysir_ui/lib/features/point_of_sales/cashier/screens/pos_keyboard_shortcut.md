# POS Keyboard Shortcuts

These shortcuts are declared in `pos_shell_shortcuts.dart` and wired through `POSShellScaffold`.

| Shortcut | Action |
| --- | --- |
| F2 | Focus product search |
| Enter in search | Add the exact or first visible product match |
| F4 | Scan or enter barcode/SKU |
| F6 | Hold the current order |
| F7 | Open held orders |
| F9 | Open payment |
| Ctrl+N | Start a new order |
| Ctrl+0 | Auto layout |
| Ctrl+1 | Counter layout |
| Ctrl+2 | Compact layout |
| Ctrl+3 | Checkout layout |
